Node commands:

node -v

npm install
npm uninstall
npm ci
npm root -g
npm list -g --depth=0
npm pkg set scripts.[任意の名前]="任意のコマンド"
※npm-scriptsのフィールドのコマンドではnode_modulesフォルダー内の実行
ファイルを呼び出し可能。そのためnpxコマンドなどを記述する必要はない。

npx 

NVM commands:
nvm --help
nvm current
nvm install <version>
nvm uninstall <version>
nvm ls
nvm ls-remote
nvm ls-remote --lts
nvm use <version>
nvm alias default <version>

オープンソースのJavaScriptの実行環境である。
