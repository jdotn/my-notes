# Git

## ファイル(またはディレクトリ)の名前変更

```bash
git mv 変更前ファイル名 変更後ファイル名
```

ファイルやディレクトリの名前を変える機会があるときはgit mvを使うべし. 名前変更前の履歴を引き継ぐことができる.

git mvを使用せずに名前を変えると, 削除->新規作成とみなされる.
